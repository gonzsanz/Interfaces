ACERCA DE

Para la realización de este proyecto se partió de un prototipo realizado mediante Figma.

En él, se parte los siguientes conceptos:

· Una pagina web estática con 6 secciones

· Un estilo minimalista a escala de grises

Finalmente, evoluciona junto estos conceptos:

· Uso exclusivo de líneas y logos como elementos visuales decorativos

· Efectos sutiles visuales sobre los elementos

Para la realización de este proyecto se ha usado:

· HTML 5

· CSS3:

Estructura:

  - Grid: Como estructura principal
  - Flex: Como estructura complementaria
  - @Media: Para restringir los diferentes tamaños de visualización

Efectos:

  - Hover
  - opacity
  - transition
  - transform
  - border: Como elemento visual decorativo
  - background-image
  - color
  - Animaciones: Keyframes

· CSS Validator:
